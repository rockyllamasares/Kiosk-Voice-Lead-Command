## Kiosk Voice Lead Command 
is an interactive ordering system similar to a Point-of-Sale (POS) kiosk that integrates voice-guided commands to assist users throughout the ordering process. The system is designed to improve accessibility, usability, and customer experience by providing audio instructions and responses, allowing users to navigate the interface even without technical knowledge. It supports product selection, order customization, and transaction processing while utilizing voice prompts as a virtual assistant.

### Key Features:

Voice-guided navigation and instructions

Product browsing and ordering interface

Add to cart and order customization

Transaction summary and confirmation

User-friendly kiosk UI

Accessibility support for first-time users

## Technology Concept:
The system combines traditional POS functionality with speech output to assist customers in making orders without needing staff assistance.
